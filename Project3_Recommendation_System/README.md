# 🎬 Project 3 - Movie Recommendation System

## 📌 Project Overview

This project is a simple rule-based movie recommendation system developed using Python. It takes user preferences, including genre and language, and recommends movies that match those preferences.

## 🎯 Key Features

- Takes user input for preferred genre
- Takes user input for preferred language
- Matches preferences using logical conditions
- Displays matching movie recommendations
- Handles cases where no exact match is found

## 🛠️ Technologies Used

- Python 3
- Visual Studio Code

## ▶️ How to Run

Run the following command:

```bash
python project3.py
```

Then enter your preferred genre and language when prompted.

### Example

```text
Enter your preferred genre: sci-fi
Enter your preferred language: english

Recommended Movies:
- Inception
- Interstellar
```

## 🧠 Recommendation Logic

The system compares the user's genre and language preferences with the movie dataset. If both preferences match a movie, that movie is added to the recommendation list.

- How to use lists and dictionaries to store data

- How to apply conditional logic
