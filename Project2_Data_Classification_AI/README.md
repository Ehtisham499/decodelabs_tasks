# 📊 Project 2 - Data Classification Using AI

## 📌 Project Overview

This project demonstrates a basic Machine Learning classification model using Python and Scikit-learn. It loads the Iris dataset, splits it into training and testing sets, trains a Decision Tree classifier, and evaluates the model's accuracy.

## 🎯 Objectives

- Load a dataset
- Split the dataset into training and testing sets
- Train a classification model
- Test the model
- Measure prediction accuracy

## 🛠 Technologies Used

- Python
- Scikit-learn
- Visual Studio Code
- Git
- GitHub

Install Scikit-learn:

